# Kinoafisha Data Governance Project

## Цель проекта

Описать и проверить качество данных датасета «Киноафиша», используемого в Power BI-дашборде для анализа 1000 лучших фильмов с сайта Киноафиши.

Проект направлен на подготовку базового Data Governance-комплекта: описание данных, бизнес-терминов, правил качества и выявленных проблем.

## Функциональность

- описание структуры датасета в словаре данных;
- описание бизнес-терминов в глоссарии;
- формирование правил качества данных;
- проверка пропусков в обязательных полях;
- проверка полных дубликатов;
- проверка дубликатов по ключевым полям;
- проверка диапазона рейтинга;
- проверка корректности года выпуска;
- проверка пустых жанров;
- проверка неоднородных категорий жанров и стран;
- фиксация найденных проблем в реестре DQ-issues.

## Файлы

- `data/kinoafisha_data.csv` — исходный датасет из Power BI-проекта
- `notebooks/dg_kinoafisha.ipynb` — Jupyter Notebook с проверками качества данных
- `main/data_dictionary.xlsx` — словарь данных
- `main/business_glossary.xlsx` — бизнес-глоссарий
- `main/dq_rules.xlsx` — правила качества данных
- `main/dq_issues_register.xlsx` — реестр выявленных проблем качества данных

## Проверки качества данных

В проекте реализованы следующие проверки:

- **Полнота** — поиск пропусков и пустых значений в полях `Название фильма`, `Жанр`, `Год`, `Рейтинг`, `Страна`;
- **Уникальность** — поиск полных дубликатов строк и повторов по ключу `Название фильма + Год + Страна`;
- **Валидность** — проверка, что рейтинг находится в диапазоне от 0 до 10;
- **Корректность года** — проверка, что год выпуска находится в допустимом диапазоне;
- **Согласованность** — проверка неоднородных категорий, лишних пробелов и различий регистра в текстовых полях.

## Результат проверок

По результатам анализа датасета выявлена одна реальная проблема качества данных: пропуск в поле `Страна` для фильма `Monsta X: Connect X in Cinemas`.

Также проверка по ключу `Название фильма + Год + Страна` выявляет повторяющиеся строки, однако они не считаются техническими дублями, так как один фильм может быть представлен несколькими строками для разных жанров.

Критичных ошибок по рейтингу, году выпуска, жанрам и полным дубликатам не выявлено.
