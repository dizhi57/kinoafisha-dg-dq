# Data Governance проект: Киноафиша

## Цель проекта

Цель проекта — описать и проверить качество данных датасета «Киноафиша», используемого в Power BI-дашборде для анализа фильмов.

Проект включает:
- описание структуры данных;
- бизнес-глоссарий;
- правила качества данных;
- автоматизированные DQ-проверки;
- реестр выявленных проблем.

## Источник данных

Данные взяты из Power BI-проекта «Киноафиша», подготовленного на основе рейтинга фильмов с сайта Киноафиша.

## Основная таблица

Таблица: `Киноафиша`

Поля:
- Название фильма
- Жанр
- Год
- Рейтинг
- Страна

## Основные проверки качества данных

Проверяются:
- пропуски в обязательных полях;
- дубликаты фильмов;
- корректность диапазона рейтинга;
- корректность года выпуска;
- пустые жанры;
- неоднородные категории жанров и стран.

## Артефакты проекта

- `data_dictionary.xlsx` — словарь данных;
- `business_glossary.xlsx` — бизнес-глоссарий;
- `dq_rules.xlsx` — правила качества данных;
- `dq_checks.ipynb` — автоматизированные проверки;
- `dq_issues_register.xlsx` — реестр проблем качества данных.
